# Credits

## Built with AI

RAG Research Workbook was designed and built entirely in collaboration with **Claude** (claude-sonnet-4, Anthropic) through an extended conversation in [Claude.ai](https://claude.ai).

The entire project — from initial concept through to the finished application — was produced through iterative dialogue with Claude, including:

- Application architecture and design
- All source code (Electron main process, preload bridge, renderer UI)
- Cross-platform support (macOS and Windows)
- GitHub Actions CI/CD pipeline for automated builds
- macOS code signing and notarization workflow
- Browser version (GitHub Pages)
- All documentation (README, DEVELOPER.md, CLAUDE_AI.md, this file)

No code was written by hand. Every file in this repository was generated, reviewed, and refined through conversation.
